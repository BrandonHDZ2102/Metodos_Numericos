# 🍏 Métodos de Integración Numérica

La integración numérica consiste en un conjunto de algoritmos para calcular el valor aproximado de una integral definida. Estos métodos son indispensables cuando la función no tiene una primitiva elemental o cuando solo se conocen valores tabulados de la función.
