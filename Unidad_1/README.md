## ⚙️ Conceptos Clave
* **Serie de Taylor:** Método para aproximar funciones mediante una suma de términos calculados a partir de sus derivadas.
* **Convergencia:** Propiedad de una serie de acercarse al valor real a medida que aumentan los términos.
* **Error de Truncamiento:** La diferencia entre el valor real y la suma parcial de la serie.
