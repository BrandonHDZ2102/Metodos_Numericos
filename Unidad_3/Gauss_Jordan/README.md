# 🔳 Método de Gauss-Jordan

El método de Gauss-Jordan es una variación de la eliminación gaussiana. La principal diferencia es que cuando se elimina una incógnita, se hace tanto en las filas inferiores como en las superiores, transformando la matriz de coeficientes en una **matriz identidad**.

## ⚙️ Diferencia con Eliminación Gaussiana
* **Eliminación Gaussiana:** Deja una matriz triangular superior y requiere sustitución hacia atrás.
* **Gauss-Jordan:** Deja una matriz diagonal (identidad), por lo que los valores de las variables se obtienen directamente sin pasos adicionales.
